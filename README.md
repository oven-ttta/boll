# 🎀 เกมแต่งตัวตุ๊กตา (Dress-Up Doll Game)

เกมแต่งตัวตุ๊กตาสไตล์น่ารัก สร้างด้วย [Phaser 3](https://phaser.io/) (เวอร์ชัน 3.90.0)
กราฟิกทั้งหมด **วาดด้วยโค้ด** (Phaser Graphics → generateTexture) จึงไม่ต้องมีไฟล์รูปภาพเลย

## ▶️ วิธีเล่น

ดับเบิลคลิกไฟล์ `index.html` ได้เลย — ไม่ต้องติดตั้งอะไร ไม่ต้องรัน server
(Phaser ถูกดาวน์โหลดไว้ใน `lib/` แล้ว เล่นออฟไลน์ได้)

| ปุ่ม | หน้าที่ |
|---|---|
| แท็บ 8 หมวด | สีผิว / ทรงผม / สีผม / เสื้อ / ท่อนล่าง / รองเท้า / เครื่องประดับ / ฉากหลัง |
| 🎲 สุ่มชุด | สุ่มทุกหมวดพร้อมกัน |
| 🔄 เริ่มใหม่ | กลับเป็นชุดเริ่มต้น |
| 📷 บันทึกรูป | เซฟภาพตุ๊กตาเป็นไฟล์ PNG |
| คลิกที่ตุ๊กตา | ตุ๊กตาเด้งดึ๋ง + หัวใจลอย 💕 |

ชุดที่แต่งไว้จะถูกจำอัตโนมัติ (localStorage) — ปิดเปิดใหม่ชุดเดิมยังอยู่

## 📁 โครงสร้างโปรเจกต์

```
index.html        ← หน้าเว็บหลัก (เปิดไฟล์นี้เพื่อเล่น)
lib/
  phaser.min.js   ← Phaser 3.90.0
src/
  main.js         ← ตั้งค่าเกม (ขนาดจอ 1280x720, Scale.FIT)
  data.js         ← ข้อมูลไอเทมทุกชิ้น (ชื่อ, สี, ธง isDress ฯลฯ)
  art.js          ← โค้ดวาดกราฟิกทั้งหมด (ตัวตุ๊กตา ผม เสื้อผ้า ฉาก UI)
  scenes.js       ← PreloadScene (สร้างเท็กซ์เจอร์) + GameScene (เกมจริง)
```

## 🧠 หลักการทำงานสำคัญ

- **เท็กซ์เจอร์ตุ๊กตาทุกชิ้นมีขนาด 420×600 พิกัดตรงกันหมด** — เสื้อผ้า ผม รองเท้า
  วางซ้อนกันเป็นเลเยอร์ตรงตำแหน่งพอดีโดยไม่ต้องคำนวณอะไรเพิ่ม
- ลำดับเลเยอร์ (หลัง → หน้า): ผมหลัง → ตัว → เปลือกตา → ท่อนล่าง → เสื้อ → รองเท้า → ผมหน้า → เครื่องประดับ
- **สีผม** ใช้เทคนิค tint: วาดผมเป็นสีขาว-เทา แล้วคูณสีด้วย `setTint()`
  จึงได้ ทรงผม × สีผม ทุกคู่ฟรีๆ โดยไม่ต้องวาดเพิ่ม
- **เดรส** (`isDress: true`) จะซ่อนท่อนล่างอัตโนมัติ และถ้าเลือกท่อนล่างขณะใส่เดรส
  เกมจะสลับเสื้อกลับเป็นเสื้อยืดให้เอง
- เท็กซ์เจอร์เสื้อผ้ามี frame ชื่อ `thumb` (ครอปเฉพาะบริเวณไอเทม) ไว้ทำรูปย่อในเมนู
  ⚠️ การเพิ่ม frame ทำให้ frame เริ่มต้นของ Phaser เปลี่ยน — เลเยอร์บนตัวตุ๊กตาจึงต้องระบุ `'__BASE'` เสมอ

## ➕ วิธีเพิ่มไอเทมใหม่ (เช่น เสื้อตัวใหม่)

1. เพิ่มชื่อใน `src/data.js` → array `TOPS`
2. เพิ่มฟังก์ชันวาดใน `src/art.js` → array `TOP_DRAWERS` (ลำดับต้องตรงกับ data.js)
   - วาดลงพิกัดตัวตุ๊กตา: ลำตัว x≈168–252, ไหล่ y≈236, เอว y≈336
   - ทำเส้นขอบด้วยแพทเทิร์น 2 รอบ: วาดสีเข้มขยายขอบ 3px ก่อน แล้ววาดสีจริงทับ
3. เสร็จแล้ว! เมนูกับระบบสุ่ม/เซฟรองรับให้อัตโนมัติ

## 🔧 โหมดทดสอบ (สำหรับนักพัฒนา)

เปิดพร้อม query string เพื่อกำหนดชุดเริ่มต้น:

```
index.html?outfit=สีผิว,ทรงผม,สีผม,เสื้อ,ท่อนล่าง,รองเท้า,เครื่องประดับ,ฉาก&tab=หมายเลขแท็บ
เช่น  index.html?outfit=2,2,3,4,0,1,4,3&tab=1
```
